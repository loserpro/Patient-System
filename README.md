# Patient System
简单系统（ASP.NET Core Web API）.net8.0开发

## 项目简介
基于 ASP.NET Core 开发的超级简单系统

## 技术栈
- 后端框架：ASP.NET Core Web API + efcore + mysql
- 开发工具：Visual Studio 2022

## 功能模块
- 信息管理、登录


## 运行方式
1. 使用 Visual Studio 打开 `Patient System.sln`
2. 在文件appsetttings.json中配置数据库连接字符串
3. 启动项目即可运行 API 服务，使用http

## 项目结构
- Controllers：API 接口控制器
- Models：实体模型
- Data：数据访问相关
- Program.cs：程序入口
